Errors Presenter

Errors presenter is a component helper with can easly deal with your form errors, works like this: Import the component right under input (or wherever you want) and pass the field control as param.

Then the component will react and show all errors in control.errors property.

You should create a object with error object key and function which returns a string. If the error have any payload, you can receive as parameter on this function. Check the Module example above.

Using Angular Material? just put fveh-errors-presenter inside the mat-error tags
