Form Is Valid Decorator

This decorator should help you with some functions that should be executed only if the form is valid (e.g. onSubmit)

Send the form name as parameter and the decorated function will be executed only if this form is valid.

Behind the scenes, decorator will be checking if controls are valid, if not, it will pass by every control (deeply) marking the fields as touched.
